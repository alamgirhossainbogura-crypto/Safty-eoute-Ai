# SafeRoute AI - Hands-Free Safety Tool
**Built for the Obserc Safety Hackathon 2026**

## Features
- **Voice Trigger:** বিপদে পড়লে "Help Help" বললেই SOS সক্রিয় হয়।
- **Real-time Location:** জিপিএস (GPS) এর মাধ্যমে সঠিক অবস্থান নির্ণয়।
- **Clean UI:** ডার্ক মোড ডিজাইন যা অবসার্ক (Obserc) অ্যাপের মতো।

## Tech Stack
- HTML5, CSS3, JavaScript
- Web Speech API (Voice Recognition)
- Geolocation API (Location Tracking)
